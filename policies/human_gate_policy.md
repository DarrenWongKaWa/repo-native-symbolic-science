# Human Gate Policy

## Overview

All operations that modify scientific state or promote claim types require explicit human authorization.

## Operations Requiring Human Gate

- IBP (Integration By Parts) authorization
- Canonical state promotion
- Scientific claim closure
- Global Guard lift
- Publishing private scientific content

## Human Gate Protocol

1. Agent prepares a structured gate request
2. Human reviews and explicitly approves or denies
3. Decision recorded as immutable amendment
4. Workflow proceeds according to authorized scope

## No Implicit Authorization

Assumptions, defaults, or inferences must not substitute for explicit human gate authorization.
If the human gate has not been cleared, the operation is blocked.
