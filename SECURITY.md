# Security Policy

## Reporting a Vulnerability

To report a security vulnerability, please open an issue on the GitHub repository with a detailed description.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Security Considerations

This framework is a tool for human-agent symbolic science. It does not:
- Execute untrusted code automatically
- Connect to external services without explicit configuration
- Store or transmit private scientific data

All computational engines operate within bounded, authorized operation sets.
