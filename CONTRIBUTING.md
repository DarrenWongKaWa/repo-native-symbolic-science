# Contributing

Thank you for your interest in contributing to Repo-Native Symbolic Science.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Development Guidelines

- Follow the repo policy in `REPO_POLICY.md`
- Use the task templates under `task_templates/`
- Run the public fixture suite before submitting: `python3 tests/engine_fixtures/run_fixture_suite.py`
- Ensure all validators pass

## Code Style

- Python: PEP 8
- JSON schemas: draft-07
- Markdown: CommonMark
